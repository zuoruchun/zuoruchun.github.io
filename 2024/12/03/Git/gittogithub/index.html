发表于 2024-12-03
  1. 打开终端并导航到当前文件夹

​ cd /path/to/your/current/folder

  1. 初始化 Git 仓库（如果还没有初始化）

​ git init

  1. 添加远程仓库

​ git remote add origin https://github.com/zuoruchun/TCCIR.git

  1. 添加所有文件到暂存区

​ git add .

  1. 提交更改

​ git commit -m "Initial commit"

  1. 推送到远程仓库

​ git push -u origin master

如果远程仓库已经存在并且有内容，你可能需要先拉取远程仓库的内容，然后再推送
